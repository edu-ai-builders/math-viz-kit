## 二、认知动作枚举说明

每个项目只选 **一个** 核心认知动作：

| 动作 | 含义 | 典型项目 |
|------|------|---------|
| 比较 | 并排观察两个或多个对象的差异 | 分数条比较器、等差等比数列比较 |
| 分解 | 将整体拆解为可见的组成部分 | 勾股定理拼图、位值积木、面积公式剪拼 |
| 变换 | 对象经历平移/旋转/映射后的变化 | 函数参数实验室、平移旋转变换器 |
| 逼近 | 通过渐进过程趋近某个值 | ε-δ极限、黎曼和、泰勒展开 |
| 累积 | 结果由多个单元叠加而成 | 积分累积量、数列求和堆叠图 |
| 缩放 | 比例关系，相似放大缩小 | 相似三角形、比例尺 |
| 投影 | 高维到低维的映射关系 | 向量投影、正交投影、PCA |
| 模拟 | 随机过程或数值迭代 | 中心极限定理、掷骰子概率、梯度下降 |

## 三、可视化收益评分指南

填写 `viz_value_score` 时，用以下问题辅助判断：

1. **"如果只有文字和公式，学生能理解吗？"** → 难 → 分高
2. **"这个概念有没有'变与不变'的动态关系？"** → 有 → 分高
3. **"学生的常见误区能被可视化直接纠正吗？"** → 能 → 加1分
4. **"课堂演示时，这个动图能让学生发出'哦！'的反应吗？"** → 能 → 加1分

## 四、Portfolio 三层组织结构

Portfolio 不按年级机械排列，而是按三层组织：

```
第一层：认知动作（顶层标签）
   比较 / 分解 / 变换 / 逼近 / 累积 / 缩放 / 投影 / 模拟

第二层：数学主题
   数与运算 / 代数 / 函数 / 几何 / 概率统计 / 微积分 / 线性代数

第三层：具体项目
   id + title + showcase_moment
```

**推荐的三项组合展示方式：**
- 同认知动作 × 不同学段（展示概念演进）
- 同主题 × 不同交互形式（展示设计多样性）
- 不同主题 × 同现实情境（展示数学建模视角）
