> 由 AI × 学习科学驱动生成，配套一套可复用的生成技能。

双击 **`数学目录.html`** 在浏览器中打开 — 即可浏览所有工具，支持网格预览和按学段筛选。

无需安装任何软件，无需网络（工具本体），无需服务器。

## 目录页功能

打开 `数学目录.html` 后：

- **网格视图（默认）**：多列缩略图，每张卡片内嵌实时预览，点击直接打开工具
- **侧栏视图**：左侧列表 + 右侧全屏嵌入预览，适合逐一浏览
- **筛选与排序**：按学段、难度、优先级、认知动作分类查看
- **懒加载**：仅加载进入视口的预览，127 个工具不卡顿

## 设计理念

每个工具背后有三个核心问题：

1. **为什么要可视化？** — 只做"静态图讲不清楚、动态一秒懂"的知识点
2. **让大脑做什么？** — 每个工具只承载**一个认知动作**（比较 / 分解 / 变换 / 逼近 / 累积 / 模拟…）
3. **反馈说什么？** — 不只显示对错，而是揭示数学规律（"无论你怎么改变三角形，a²+b²始终等于c²"）

这套设计规则来自认知负荷理论（Sweller）、多表征研究（Frontiers in Psychology）、具身认知（Abrahamson）等学习科学文献，完整版见 `system/教学法规则库-v1.md`。

## 用 AI 生成新工具

本系列配套了一个 AI 技能，可以从"知识点名称"一步生成 HTML 工具。

**调用方式（在 Claude 中）：**

```
/数学可视化
模式: FULL
知识点: 复利与指数增长
学段: 高中
```

三个模式：

| 模式 | 输出 |
|------|------|
| `EVAL` | 判断是否值得可视化 + 推荐交互形式 |
| `PRD` | 完整产品设计文档（结构、交互、布局）|
| `CODE` | 可直接运行的 HTML 文件 |
| `FULL` | 三者依次输出 |

详细规范见 **`数学可视化-skill.md`**。

## 技术规格

- **技术栈：** 原生 HTML + CSS + JavaScript（Canvas 2D / SVG），零构建工具
- **字体：** Noto Sans SC + JetBrains Mono（Google Fonts，离线也有降级字体）
- **兼容性：** 任何现代浏览器，包括手机浏览器
- **离线使用：** 工具本体完全离线可用；仅字体需要网络（降级不影响功能）
