# 数学可视化 · AI 生成技能 v1

> 将任意数学知识点 → 教学评估 → 设计文档 → 可运行 HTML 可视化工具

---

## 触发格式

```
/数学可视化
模式: EVAL | PRD | CODE | FULL
知识点: [名称]
学段: 小学 | 初中 | 高中 | 大学
年级范围: [可选，如"初一下"、"高二"]
特殊要求: [可选]
```

**模式说明：**

| 模式 | 输入 | 输出 |
|------|------|------|
| `EVAL` | 知识点 + 学段 | 元数据 YAML + 可视化评估 + 教学法建议 |
| `PRD` | EVAL 结果（或自动先执行 EVAL） | 完整结构化设计文档 |
| `CODE` | PRD 结果 | 可在浏览器直接运行的自包含 HTML 文件 |
| `FULL` | 知识点 + 学段 | 依次输出 EVAL → PRD → CODE |

---

## EVAL 模式

**目标：** 判断这个知识点是否值得做成可视化，并推荐最佳方案。

### 输出格式

````markdown
## EVAL：[知识点名称]

### 元数据
```yaml
title: ""
stage: ""
grade_range: ""
topic_area: ""
concept_name: ""
math_essence: ""
core_misconception: ""
viz_value_score: 0        # 1–5，见评分标准
dynamic_score: 0
interaction_necessity: 0
cognitive_action: ""      # 比较|分解|变换|逼近|累积|缩放|投影|模拟
primary_interaction: ""   # 滑杆调参|拖拽操作|点击探索|步骤推进|输入验算|模拟实验|拼图重组
open_source_priority: ""  # A|B|C
```

### 可视化评估
**值得做吗？** [是 / 谨慎 / 不推荐]

**理由：**
- 核心动态关系：[一句话描述"变与不变"]
- 纠正的主要误区：[学生最常犯的错误]
- 静态图是否足够：[是/否，理由]

### 教学法建议
适用规则：**[V1/V3/V5 等]**

| 规则 | 如何在本项目中应用 |
|------|--------------------|
| Vn 规则名 | 具体操作 |

### 推荐的认知动作
`[认知动作]` — [一句话解释为什么这个动作最合适]
````

### 输出格式

````markdown
## PRD：[知识点名称]

### 一句话定位

- **核心知识点：** [概念名]
- **学段 / 年级：** [学段，年级范围]
- **主要误区：** [学生最常见的错误理解]
- **认知动作：** [单一认知动作名]

### 为什么要可视化
- **[规则 V1]：** [这个知识点的"变与不变"是什么]
- **[规则 Vn]：** [其他适用规则的理由]

### 页面结构

#### 初始状态
[打开页面时，用户看到什么。应尽量简洁——V3 原则]

#### 主要交互流程
1. [第一步：用户操作]
2. [第二步：系统响应]
3. [以此类推]

#### 模块布局
| 区域 | 内容 | 位置 |
|------|------|------|
| 主展示区 | [核心图形/动画] | 中央 |
| 控制区 | [滑杆/按钮] | [位置] |
| 反馈区 | [概念反馈文字] | 底部 |

### 交互细节
- **交互类型：** [滑杆 / 拖拽 / 步骤推进 / ...]
- **核心参数：** [参数名，范围，默认值]
- **动画：** [是/否，动画描述]
- **概念反馈触发：** [在什么操作后显示什么文字]

### 技术规格

- **字体：** Noto Sans SC + JetBrains Mono（Google Fonts）
- **色板：** 暖奶油系（见视觉规范）
- **响应式：** [单列 / 双列 / 三列，手机适配方式]
- **单文件可行：** 是

### MVP 范围
**包含：**
- [功能1]
- [功能2]

**不包含（后续版本）：**
- [功能]
````

## CODE 模式

**目标：** 生成完整、可在浏览器直接运行的自包含 HTML 文件。

### 强制规范

#### 视觉风格
```css
:root {
  --bg: #f7f6f2;          /* 暖奶油背景 */
  --surface: #fff;        /* 卡片/面板白色 */
  --ink: #1a1814;         /* 主文字近黑 */
  --ink2: #6b6860;        /* 辅助文字灰 */
  --border: #e4e2da;      /* 边框暖灰 */
  --blue: #2a5c8f;        /* 主强调色（交互元素）*/
  --blue-light: #e8f0f8;
  --teal: #0f6e56;        /* 第二强调色（正确/结果）*/
  --teal-light: #e1f5ee;
  --amber: #854f0b;       /* 第三强调色（标注/辅助）*/
  --amber-light: #faeeda;
  --radius: 12px;
  --radius-sm: 8px;
}
```

#### 必要的 meta 标签（用于目录索引）
```html
<meta name="grade" content="[小学|初中|高中|大学]">
<meta name="topic" content="[主题名称]">
<meta name="difficulty" content="[低|中|高]">
<meta name="priority" content="[A|B|C]">
<meta name="cognitive" content="[认知动作]">
<meta name="desc" content="[一句话描述，约30字]">
```

#### 教学法注释（必须出现在 JS 顶部）
```js
// PEDAGOGY V1 动态跃迁: [核心动态关系描述]
// PEDAGOGY V3 分步揭示: [揭示策略描述]
// PEDAGOGY V8 概念反馈: [反馈文字策略描述]
// [其他适用规则...]
```

#### 结构模板
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>[工具名] · 数学可视化</title>
  <!-- meta 标签（见上方）-->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;500;700&family=JetBrains+Mono:wght@400;700&display=swap" rel="stylesheet">
  <style>
    /* 色板变量 + 组件样式，全部内联 */
  </style>
</head>
<body>
<header>
  <div class="badge">[学段] · [主题] · 认知动作：[动作名]</div>
  <h1>[工具名]</h1>
  <p class="sub">[副标题，一句话说明核心体验]</p>
</header>
<main>
  <!-- 控制区 -->
  <!-- 主展示区（Canvas 或 SVG） -->
  <!-- 反馈区（.fb 元素）-->
</main>
<script>
  // PEDAGOGY 注释
  // 实现代码
</script>
</body>
</html>
```

### 质量检查清单

CODE 模式输出前，Claude 必须自检以下项目：

- [ ] `<meta name="grade">` 等 6 个 meta 标签均已填写
- [ ] JS 顶部有 `// PEDAGOGY` 注释
- [ ] 打开页面时不超过 3 层信息（V3）
- [ ] 只有 3 种强调色（V4）
- [ ] 有重置按钮或重新开始入口（V3）
- [ ] 反馈文字包含数学规律（V8），不只是 ✓✗
- [ ] 所有 CSS 和 JS 内联，无外部依赖（Google Fonts 除外）
- [ ] 在 375px 宽度下不出现横向滚动条

## 认知动作速查

| 动作 | 含义 | 典型知识点示例 |
|------|------|----------------|
| 比较 | 并排观察多个对象的差异 | 分数大小、数列增长速率、函数族 |
| 分解 | 将整体拆解为可见部分 | 勾股定理面积证明、位值积木、积分面积 |
| 变换 | 对象经历映射后的变化 | 函数参数、线性变换、平移旋转 |
| 逼近 | 渐进过程趋近某值 | ε-δ极限、黎曼和、泰勒展开 |
| 累积 | 结果由多个单元叠加 | 积分、数列求和、概率分布 |
| 缩放 | 比例关系与相似放大 | 相似三角形、比例尺、对数坐标 |
| 投影 | 高维向低维映射 | 向量投影、PCA、三视图 |
| 模拟 | 随机过程或数值迭代 | 中心极限定理、梯度下降、马尔可夫链 |
